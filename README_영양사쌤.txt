1.코드 수정을 위한 파일 오픈은 user_checker.py 파일을 
  우클릭->Edit with IDLE->Edit with Idle 3.8로 오픈합니다

2.코드 파일에서 수정가능한 부분은 명단에 기록된 기호입니다
  상단 User Interface에서 수정가능합니다.
  기본적으로 카드지참 시 O, 카드미지참 시 A를 기록하게 되어있는데, 자유롭게 변경하셔도 됩니다
  (글자수 및 형식 제한 없음)
  쌍따움표 안에 적힌 초록색 기호들을 바꾸고 파일을 저장(ctrl+S)하시면 됩니다.

3.[가장중요] 명단 파일 이용법
  프로그램 사용시 user_list.csv 파일에 모든 기록이 저장됩니다.
  이 파일은 월초에 user_list_new.csv 파일을 복사하여 사용되도록 구성되었습니다.
  따라서 초기에 user_list_new.csv 파일을 잘 초기화하여 user_ckecker.py가 있는 폴더에 같이 저장해두어야 합니다
  초기화해야하는 형식은 첫행은 열 설명을 위한 칸으로 첫행에는 이용자를 적으면 안됩니다.
  이후 두번째 행부터 이용자의 번호와 이름을 적으시면 됩니다.
  이때 중간에 빈칸이 있어도 상관없고, 이용자 번호의 순서가 뒤섞여 있어도 상관없으니 편하게 두어도 됩니다.
  즉, 초기화 파일은 N*2 형태로 구성되어야 하며 새 excel파일을 만들어서 초기화 파일을 만드는 걸 권장합니다.
  
  파일을 만드신 후에는 "꼭" csv 확장자로 저장하여야합니다
  다른 이름으로 저장-> 파일이름은 user_list, 파일형식은 csv(쉼표로 분리)를 선택하시고 저장하시면 됩니다.

  초기화 해놓으시면 한달간의 사용 후 기록이 저장된 파일을 가져오시면 됩니다.

  이용중간에 csv파일에 이용자를 하단에 추가하시거나 순서를 바꾸시거나 수기로 표에 기록을 하시는건 
  전혀 문제없습니다.

4.명단이 변경될 경우 하셔야 할 일을 요약하면
  [1]명단으로 user_list_new.csv를 만드시고
  [2]프로그램 폴더에 원래 user_list_new.csv를 덮어 씌우시고
  [3]기록하는 사회복무요원에게 명단이 변경됐으니 초기화를 해달라! 고 하시면 됩니다
