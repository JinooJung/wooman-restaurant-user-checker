1. user_checker.py 를 우클릭->Edit with IDLE->Edit with IDLE 3.8-> F5로 실행
  (F5가 작동하지 않으면 Run->Run Module)
  첫번째 실행시 시간이 살짝 걸리니 기다릴것
  재실행 할때는 열린 user_checker.py에서 F5만 누르면 됨. IDLE Shell에서 F5치는거 아님!

2. 프로그램이 시키는대로 하면	
3. 식당 이용이 시작되면
  -카드 지참시 번호만 입력
  -카드 미지참시 성함 입력 or 번호를 알 경우 숫자앞에 0을 붙쳐 입력 ex) 083
  -잘못 입력하면 숫자앞에 -를 붙쳐 입력 ex)-83
  -끝 이라고 치면 파일 저장하면서 끝냄
  -저장 이라고 치면 파일 저장됨. 끝이나 저장안하면 열심히 기록한거 excel파일에 반영안되니 "꼭" 해줄것

5.[중요한점] 프로그램을 사용하고 있을때는 user_list.csv 절대 오픈하지 말것!!!!!!
  저 파일 열고싶으면 프로그램 끝이라고 쳐서 끝내고 오픈해서 확인후 엑셀 닫고 다시 프로그램 실행할것
